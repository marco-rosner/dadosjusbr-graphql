# DadosJusBr GraphQL

This project is a GraphQL server to query the data from DadosJusBR. Using Apollo Server, Express and Type GraphQL

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run start
```
